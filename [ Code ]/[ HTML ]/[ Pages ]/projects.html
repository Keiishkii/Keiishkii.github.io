Open GL Game Engine - C++

This project involved the creating of a fully functioning, but basic, game engine. Within which, I had attempted to create demo's of levels from Xenoblade Chronicles and Metroid Prime. Unfortunately I did not get animation for the FBX format working and neither the implementation of a physics engine such as PhysX. However, the project did support custom script writing, skybox, PBR shaders and trigger colliders. And as a showcase of running around firing missiles and exploring the levels it worked out quite well.
