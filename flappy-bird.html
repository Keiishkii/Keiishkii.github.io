Flappy Bird Clone ~ Project ~

I made this during college. It’s a small Flappy Bird clone which just demonstrates some more interesting ways in which I could animate. These being the scrolling background, moving pistons and the player’s animations themselves. The game uses simple physics and some basic box collisions on the player and obstacles.

The approaches used in the project included.

  • JFrame and JComponent use
  • Input Controllers
  • Animation Controllers
  • Audio Controllers
  • Loading maps from files
