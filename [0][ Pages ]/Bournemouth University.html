I am currently employed as a Research Assistant within the MINE Cluster at Bournemouth University, my work typically involves the development of solutions for experiments or research projects.

EmBodied-N170 is a research project currently running within the cluster. My role on this project was to develop the experiment within Unity, this required modeling and building the scene's environment, programming the experiments game-flow and the transmission and collection of experiment data between multiple devices on a local network.

BAB Football is another project I am currently working on. The project involved the implementation of pre-recorded live game data of real football matches and replaying the game within a virtual environment, with the aid of VR. My role was the integration of a pre-trained PFNN motion matching model in which would use the input data as a way of generating each player's trajectories.
