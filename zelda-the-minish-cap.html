The Legend Of Zelda: The Minish Cap ~ Project ~

The aim of this project was to reproduce the gameboy game in C++ using the SDL libuary. All code was oringinal and only the sprites and audio where reused, these where found on 3rd party fan sites.

The aproaches used in the project included.

  • Finite state machines
  • Trigger, Damage and Solid Colliders
  • Input Controllers
  • Animatoin Controllers
  • Audio Controllers
  • Loading saves from files
