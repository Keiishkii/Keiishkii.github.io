Explorer ~ Project 2 ~

Explorer II is a project written in Java, it’s a small sandbox style game where the player is able to harvest resources from tiles to build more tiles. The game used Java's JFrame and JComponents as well as several different core libraries within the language.

The approaches used in the project included.

  • JFrame and JComponent use
  • Input Controllers
  • Animation Controllers
  • Audio Controllers
  • Loading maps from files
